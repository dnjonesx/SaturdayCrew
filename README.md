# SaturdayCrew
This is a project that Emily, Violet and I will work on.

We are thinking that we will use Python to track a free weather API that collects data and puts it into an SQL database and is showcased by a Javascript frontend. 

In order to operate with the API, you will need to create a separate venv within this folder by running python -m venv venv source venv/bin/activate on linux/mac or python -m venv venv source venv\Scripts\activate. Then you will need to run pip install -r requirements.txt.

There will be a .gitignore file that will include the venv so that it is not shared via git.
